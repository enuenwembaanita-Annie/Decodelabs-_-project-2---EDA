# Decodelabs-_-project-2---EDA
Explorative data analysis (EDA) on sales data using powerbi_ Decodelabs internship 
## Project Overview
This project performs an Exploratory Data Analysis on a sales dataset containing products such as monitors, phones, tablets, chairs, printers, and laptops to uncover patterns, trends, and outliers.

## Key Requirements
- Calculate basic statistics (Mean, Median, Count)
- Identify trends and outliers
- Summarize key observations

## Tools Used
- Power BI

## Key Findings
- Chair generates the highest revenue of $195,620
- Cancelled orders are the highest with 250 orders
- Online payment is the most preferred with 258 transactions
- Instagram is the strongest referral source with 259 referrals
- Mean order value: $1,050
- Median order value: $823.62
- Total orders: 1,200
